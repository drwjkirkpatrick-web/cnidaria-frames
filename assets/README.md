# Assets Directory

This directory contains image assets for the Cnidaria Frames project.

## Icons

- `icon-192.png` - 192x192 icon for mobile home screens
- `icon-512.png` - 512x512 icon for mobile home screens

## Creating Icons

To create these icons, you can use any image editing software to create a jellyfish silhouette or use a simple circle with a jellyfish color gradient.

For a quick placeholder, you can create simple colored squares:

```bash
# Create placeholder icons (requires ImageMagick)
convert -size 192x192 xc:#6496ff icon-192.png
convert -size 512x512 xc:#6496ff icon-512.png
```
